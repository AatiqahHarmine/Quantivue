# Quantivue
A web-based application to track income, expenses, and spending categories with interactive visual analytics. This project helps users understand their financial behavior through a clean dashboard and real-time data insights.  

Features:  
Track Income and Expenses  
Category-based Expense Tracking (Food, Grocery, etc.)  
Financial Overview Dashboard  
Pie Chart for Category-wise Spending  
Bar Chart for Income, Expenses & Balance  
Real-time Data Fetching using APIs  
Access across devices (LAN support)    

Tech Stack:  
Backend: Python, FastAPI  
Database: PostgreSQL  
ORM: SQLAlchemy  
Frontend: HTML, CSS, JavaScript  
Charts: Chart.js
